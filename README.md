# 🎀 Pokemon Ribbon Tracker

ポケモンのリボンコレクションを追跡するためのWebアプリケーション！複数世代に渡るリボン収集を簡単に管理できます。

## 📋 概要

Pokemon Ribbon Trackerは、ポケモンシリーズの各世代で獲得できるリボンを追跡・管理するためのツールです。各ポケモンに対して獲得可能なリボンをリストアップし、進捗状況を視覚的に確認することができます。

「リボン王」と呼ばれる全リボン獲得への道のりをサポートします！

## ✨ 機能

- **ポケモン検索**: 名前やIDでポケモンを検索できます
- **マイポケモン管理**: 複数のポケモン個体をニックネーム・出身ゲームとともに登録、個体ごとに進捗を独立管理
- **リボン管理タブ**: 世代別・種類別フィルタ付きリボン一覧。チェックとガイドをその場で確認
  - 折りたたみフィルタ（アクティブ件数バッジ付き）
  - Lv.制限リボンの警告バナー（チャンピオンリボン前に取るべきリボンを明示）
  - リスト表示 / チャート表示の切り替え
- **リボン取得チャート**: フェーズ別の取得ロードマップ
  - Phase 1（赤）: レベル制限ありのリボン（最優先）
  - Phase 2（黄）: コンテストリボン
  - Phase 3（緑）: ストーリー・バトル・その他
  - 取得不可リボン（このポケモンでは取れない理由を表示）
  - 進捗バーは「取得可能数」を分母に（FR/LG なら2個が上限と正しく計算）
- **転送ガイド**: 第3世代リボン取得推奨順序 + 世代間の転送方法を解説
- **進捗管理**: ブラウザのローカルストレージで進捗を保存。エクスポート/インポート対応
- **トースト通知**: インポート成功・失敗などをUIに自動消滅する通知で表示
- **2カラムレイアウト**: デスクトップではポケモン情報を左に固定しながらリボン管理できる
- **オンボーディング**: 初回アクセス時に使い方の3ステップガイドを表示

## 🚀 セットアップ方法

```bash
# リポジトリのクローン
git clone https://github.com/boitoshi/pokemon-ribbon-tracker.git
cd pokemon-ribbon-tracker

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番用ビルド
npm run build
npm run start
```

## 📱 使い方

1. ポケモンを検索して選択します
2. 「マイポケモンに登録」ボタンで個体情報（ニックネーム・出身ゲーム）を登録します
3. **リボン管理タブ**でリボンをチェックして進捗を記録します
   - フィルタ・検索でリボンを絞り込み、アコーディオンで獲得方法を確認
   - 「チャート」ビューに切り替えると取得推奨順序（フェーズ別）を確認できます
4. 「転送ガイド」タブで世代間の転送方法を確認します
5. 複数のポケモンを登録している場合は、マイポケモンパネルで切り替えながらそれぞれの進捗を管理できます

## 🔄 世代間転送経路

ポケモンの世代間転送経路は以下の通りです：
- **第3→第4世代**: パルパーク
- **第4→第5世代**: ポケシフター
- **第5→第6世代**: ポケムーバー → ポケバンク
- **第6→第7世代**: ポケバンク
- **第7→第8世代**: ポケバンク → ポケモンHOME
- **第8→第9世代**: ポケモンHOME
